# Prompt: Build a Complete MIVA Student Portfolio Website

Act as a senior frontend engineer and UI/UX designer.

Build a complete multi-page responsive student portfolio website for a COS106 Web Development Lab Assessment using only HTML5, CSS3, and Vanilla JavaScript.

Do not use Bootstrap, React, Vue, Angular, Tailwind CSS, jQuery, or any frontend framework.

The project should look modern, professional, and suitable for a university assessment.
