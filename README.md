# 🎓 MIVA Student Portfolio

A modern, fully responsive student portfolio website developed as a COS106 Web Development Lab Assessment for MIVA Open University.

The project demonstrates proficiency in HTML5, CSS3, and JavaScript by building a complete multi-page website featuring responsive layouts, animations, JavaScript interactivity, Local Storage, CRUD operations, DOM manipulation, and client-side form validation.

# 📚 Project Overview

The MIVA Student Portfolio is a personal academic website created to showcase a student's profile, projects, technical skills, and web development knowledge.

The website was built using only:

- HTML5
- CSS3
- Vanilla JavaScript

No frameworks or libraries such as React, Bootstrap, Vue, or Tailwind CSS were used.

The project satisfies the COS106 assessment requirements by demonstrating semantic HTML, responsive web design, JavaScript programming, DOM manipulation, Local Storage, form validation, and dynamic user interaction.
